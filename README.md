# Professional-Certificates-List

A curated portfolio of professional certifications, diplomas, and achievement records for Đinh Hoàng Sáng.

## Features
- **Cloud Computing**: AWS, Azure, and Google Cloud certificates.
- **AI & Data Science**: Coursera, DeepLearning.AI, and university certifications.
- **Software Engineering**: Practical training and industry-recognized credentials.

## Project Structure
```
├── README.md
├── LICENSE
└── media/              # High-resolution digital copies of certificates
```

## Showcase
Digital copies of all certificates are organized within the `media/` directory. These include milestones in Cloud, AI, and Software Engineering.
